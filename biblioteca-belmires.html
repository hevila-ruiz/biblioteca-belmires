Como buscar um livro?

Clique na aba 🔍 Buscar e digite no campo de busca qualquer parte do título ou do autor. A lista vai filtrando enquanto você digita.

Para ver só livros de um local específico (escritório, sótão, etc), use o filtro abaixo da busca.

Como adicionar um livro novo?

Clique na aba ➕ Adicionar livro, preencha pelo menos o título e escolha o local, depois clique em Salvar livro.

Como salvar as alterações?

Sempre que você adicionar, editar ou excluir um livro, aparece uma barra amarela no topo avisando "Você fez alterações".

Clique no botão 💾 Salvar arquivo. Vai baixar um novo arquivo biblioteca-belmires.html na pasta de Downloads.

Substitua o arquivo antigo pelo novo (no Google Drive, no computador, onde estiver guardado).

Importante: Se você fechar sem salvar, perde as alterações.

Como funciona com várias pessoas usando?

Quem cadastra os livros é a "guardiã" do arquivo. Os outros podem abrir e consultar à vontade.

Se mais de uma pessoa cadastrar livro novo ao mesmo tempo, um vai sobrescrever o outro. O combinado é: só uma pessoa cadastra (geralmente quem está organizando).

Os outros, quando quiserem adicionar um livro, falam com a pessoa responsável.

Onde os dados ficam guardados?

Os dados ficam dentro do próprio arquivo HTML. Quando você clica em "Salvar arquivo", ele baixa uma versão nova com todos os livros atualizados.

Por isso, é importante guardar sempre a versão mais recente. Recomendamos guardar no Google Drive (ou similar) para ter cópia de segurança.

Posso usar no celular?

Sim! Abra o arquivo direto pelo Google Drive (ou de onde estiver guardado) no celular. Funciona em iPhone (Safari) e Android (Chrome).

Para consultar é perfeito. Para cadastrar muitos livros, é mais confortável no notebook.

⚠️ Apagar tudo

Apaga todos os livros do arquivo. Não tem como desfazer. Só use se quiser começar do zero.
